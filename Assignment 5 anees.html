# Finding the best number of neighbors using the elbow method
error_rate = []
for i in range(1, 40):
    knn = KNeighborsClassifier(n_neighbors=i)
    knn.fit(X_train, y_train)
    pred_i = knn.predict(X_test)
    error_rate.append(np.mean(pred_i != y_test))

# Plotting the elbow graph
plt.figure(figsize=(12, 6))
plt.plot(range(1, 40), error_rate, color='black', linestyle='solid', marker='o', markerfacecolor='red', markersize=10)
plt.title('Error Rate vs. K Value')
plt.xlabel('K Value')
plt.ylabel('Error Rate')
plt.show()

a) What’s the best number of neighbors? Show the elbow graph.

Ans) The lowest error rate appears to be around the K value of 23, after which the error rate starts to increase or remains roughly constant with some fluctuations.

Therefore, based on this elbow graph, the best number of neighbors for the KNN classifier appears to be around 23, as this is the point with the lowest error rate before the error rate begins to increase or varies inconsistently.

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import GaussianNB
from sklearn.metrics import accuracy_score, confusion_matrix
import seaborn as sns
import matplotlib.pyplot as plt

df = pd.read_csv('Data-Week4.csv')
# Select features and target
X = df[['Attack', 'Defense']] # Features
y = df['Generation'] # Target

# Splitting dataset into training and testing set (10%)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.1, random_state=42)

# Initialize Gaussian Naive Bayes
gnb = GaussianNB()

# Fit the model on the training data
gnb.fit(X_train, y_train)

# Predicting the Test set results
y_pred = gnb.predict(X_test)

# Calculate accuracy
accuracy = accuracy_score(y_test, y_pred)
print(f'Accuracy of GaussianNB classifier: {accuracy:.2f}')

# Compute confusion matrix
cnf_matrix_simulated = confusion_matrix(y_test, y_pred)

# Plot confusion matrix using seaborn
plt.figure(figsize=(10,8))
sns.heatmap(cnf_matrix_simulated, annot=True, fmt='d', cmap='coolwarm')
plt.title('Confusion Matrix for GaussianNB')
plt.xlabel('Predicted Label')
plt.ylabel('True Label')
plt.show()

Accuracy of GaussianNB classifier: 0.19
